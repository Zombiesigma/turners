# **App Name**: KaryaHarmoni

## Core Features:

- Portfolio Display: Showcase Guntur Padilah's writings, paintings, and development projects in a visually appealing and organized manner.
- Theme Toggle: Enable users to switch between light and dark themes for optimal viewing experience.
- AI Chat Assistant: Integrate an AI-powered chat assistant (Litera AI) to provide users with information and assistance regarding Guntur Padilah's works and expertise. This feature will act as a tool, offering helpful and accurate information when the situation requires.
- Contact Form Submission: Allow users to submit inquiries and collaboration requests through a contact form that sends email via EmailJS.
- Certificate Verification: Implement a certificate verification system using a unique code and blockchain to confirm the authenticity of Guntur Padilah's certifications. Placeholder interaction to give the user a positive affirmation of a succesful verification
- 3D Background Animation: Add an engaging 3D background animation to enhance the visual appeal and create a modern, interactive user experience. A canvas-based animated background that draws the user in.
- Stats Animation: Animate counters on stats items to emphasize skills with numerical stats to bring an emphasis to the skill values.

## Style Guidelines:

- Primary color: Gold (#FFD700) to reflect Guntur Padilah's accomplishments and artistic nature. Hex: #FFD700
- Background color: Very dark gray (#121212) to provide a sophisticated and visually appealing backdrop. Hex: #121212
- Accent color: Warm amber (#FF7E00), to evoke a sense of Indonesian heritage. Hex: #FF7E00
- Headline font: 'Playfair', a serif font, is perfect to use for the various headlines to align to the original proposal. Body font: 'Inter', a sans-serif font, is clean and modern.
- Monospace font: 'Source Code Pro' will be used to format code snippets in the development section, making them highly readable.
- Font Awesome icons will be extensively used throughout the site to enhance visual appeal and provide intuitive navigation cues.
- Subtle animations such as fade-ins, slides, and scaling effects will be used on various page elements to enhance the user experience without being intrusive.
